# 📊 Customer Shopping Behavior Analysis

An **End-to-End Data Analytics Project** that analyzes retail customer purchasing behavior using **Python, SQL (PostgreSQL), and Power BI** to generate meaningful business insights.

This project simulates a **real-world analytics workflow**, starting from **data cleaning and exploration to SQL-based business analysis and dashboard visualization**.

---

# 🎯 Project Objective

The goal of this project is to analyze **customer shopping patterns, demographics, product preferences, and subscription behavior** to help businesses make **data-driven strategic decisions**.

---

# 📂 Dataset Overview

The dataset contains **transactional retail data from 3,900 purchases across multiple product categories**.

### Dataset Details

- Rows: **3,900**
- Columns: **18**

### Key Features

**Customer Demographics**
- Age
- Gender
- Location
- Subscription Status

**Purchase Details**
- Item Purchased
- Category
- Purchase Amount
- Season
- Size
- Color

**Shopping Behavior**
- Discount Applied
- Promo Code Used
- Previous Purchases
- Frequency of Purchases
- Review Rating
- Shipping Type

⚠️ Missing Data:  
37 missing values found in the **Review Rating** column.

# 🔎 Project Workflow

## 1️⃣ Data Cleaning & Preparation (Python)

- Loaded dataset using **Pandas**
- Performed initial exploration using **df.info() and df.describe()**
- Handled missing values in **Review Rating using median values**
- Standardized column names to **snake_case**
- Created new features:
  - `age_group`
  - `purchase_frequency_days`
- Removed redundant columns like `promo_code_used`
- Loaded cleaned data into **PostgreSQL database**

---

# 📊 SQL Business Analysis

Using PostgreSQL, several **business questions were analyzed**.

### Key Business Questions

1️⃣ Revenue by Gender  
Compared total revenue generated by **male vs female customers**

2️⃣ High Spending Discount Users  
Identified customers who **used discounts but spent above average**

3️⃣ Top Rated Products  
Found **top 5 products with highest average review ratings**

4️⃣ Shipping Type Comparison  
Compared **average purchase amount between Standard and Express shipping**

5️⃣ Subscribers vs Non-Subscribers  
Analyzed differences in **average spend and total revenue**

6️⃣ Discount Dependent Products  
Identified products with **highest percentage of discounted purchases**

7️⃣ Customer Segmentation  
Segmented customers into:
- New Customers
- Returning Customers
- Loyal Customers

8️⃣ Top Products per Category  
Found **top purchased products in each category**

9️⃣ Repeat Buyers & Subscriptions  
Analyzed whether **frequent buyers are more likely to subscribe**

🔟 Revenue by Age Group  
Calculated **revenue contribution across different age groups**
