# Customer-Behavior-Analysis
📊 Customer Shopping Behavior Analysis

An End-to-End Data Analytics Project that analyzes retail customer purchasing behavior using Python, SQL (PostgreSQL), and Power BI to generate actionable business insights.

This project simulates a real-world analytics workflow used by data analysts in companies, starting from data cleaning and exploration to SQL-based business analysis and interactive dashboard reporting.

🎯 Project Objective

The goal of this project is to analyze customer shopping patterns, demographics, product preferences, and subscription behavior to help businesses make data-driven strategic decisions.
